this project was inspired by the difficulty to have multiple .env across all my projects (35+),  each configured but mostly some share same environment values such as 'gemini API keys, stripe keys, redis, and many more'.  so i wish to have a central state for environments that leaves inside the root/home/<username>/  of my linux(mostly wsl2) where all my projects can just query and use.  these environments will mostly be for backends with reasonable sensitive keys. The api keys for the frontends will be stored in the frontend projects themselves. where api keys for LLMs is the reason why i started this project such that my projects can easily switch between different LLM providers and models and apikeys without them even noticing unless it is required.

Also, i want to be able to use this project as a library in my other projects to easily manage environments.
